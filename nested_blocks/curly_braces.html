波括弧¶

左波括弧（{）は，その宣言と同じ行に置きます。

def foo = {
  ...
}

技術的には，ScalaのパーサはGNUスタイルの，つまり，宣言の次の行に左括弧を配置する記法をサポートします。しかし，Scalaパーサのセミコロン推論の実装方法の制限から，このスタイルで書かれた場合には推論がうまくできません。上述の規約に従うことで，悩みの種は殆ど無くなります。
