オブジェクト¶

オブジェクトは，クラスの命名規約（大文字始まりのキャメルケース）に従います。ただし，パッケージを模したオブジェクトは除きます。かなり稀なケースですが，まさにこれが必要となる場合があります。

object ast {
  sealed trait Expr

  case class Plus(e1: Expr, e2: Expr) extends Expr
  ...
}

その他全てのケースでは，オブジェクトはクラスの命名規則に従います。
