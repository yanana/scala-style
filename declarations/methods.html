メソッド¶

メソッドは次のパターンに従って宣言するべきです。

def foo(bar: Baz): Bin = expr

ただし，Unit型を返すメソッドはこのルールへの反例となります。混乱を防ぐため，シンタックスシュガーを用いて次の様に書くべきです。

def foo(bar: Baz) {       // 戻り値の型はUnit
  expr
}
