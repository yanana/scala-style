インデント
==========

インテントは「2スペース規約」に従うべきです。したがって，次の様にするのでは無く，
:

    // 誤り！
    class Foo {
        def bar = ...
    }

次の様にインデントするべきです。 :

    // 正しい！
    class Foo {
      def bar = ..
    }

Scalaではネストしたスコープと論理ブロックが驚くほど多用されます（関数値など）。積極的に新しいブロックを作ってください。
Java出身の方にはこのスタイルにはあまりなじみがないかもしれませんが，非常に有用です。
