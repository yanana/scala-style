関数値¶

呼び出し時に役立つ，型推論の特殊なケースを関数値はサポートします。

val ls: List[String] = ...
ls map { str => str.toInt }

宣言しようとしている関数値の型をScalaが既に知っていれば，パラメータに型注釈を付与する必要はありません（上記例の場合，strが該当）。これは非常に便利なので，最大限活用しましょう。ただし，関数値を対象とする暗黙の型変換は，この型推論を無効にする事に注意しましょう。
